What is QuickProbe?

QuickProbe is a Windows desktop application built for IT administrators who need to quickly check server health and connect to remote machines. It uses WinRM (PowerShell Remoting) for Windows servers and SSH for Linux hosts.

Quick Start

Get up and running in under 5 minutes.

Getting Started
1

Download and install

Download the installer from GitHub Releases. Run it‚Äîno admin rights needed. QuickProbe installs to your user profile.

2

Enter your credentials

On first launch, enter your domain credentials (e.g., DOMAIN\username). These are stored securely in Windows Credential Manager.

3

Add your servers

Add servers manually, or scan your Active Directory domain to import them automatically. Set the OS type (Windows/Linux) for each host.

4

Monitor and connect

Click "Refresh" to probe your servers. Double-click any card to launch RDP. Use the dashboard filters to find servers with issues quickly.
